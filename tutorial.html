Print in PRINTU like a PRO with these steps:

CHOOSE FILE

Upload the PDF file you want to print, wait for it and then submit it.

CUSTOMIZE YOUR 
PRINT SETTING

Set the number of copies, the color profile and the number of pages per sheet for your transaction.

PAY AS YOU ORDER

You will be redirected to a automated payment through your GCASH, pay there to process you transaction.

SAVE YOUR QR CODE

You will be given a Quick Response (QR) code that corresponds to your transaction in our database.

SCAN THE CODE

At the printing Kiosk , scan the provided QR code at the scanning tray, place your phone and wait for the system to accept the code.

PRINT

As soon as the system accepts your code, it will then immediately print your file and your transaction completes.
